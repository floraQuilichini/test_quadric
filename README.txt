Pour générer un projet visual studio à partir de ces fichiers
-Créer un projet vide sous visual et ajouter les fichiers .cpp dans "fichiers sources" et les fichiers .h dans "fichiers d'en-tête"
- Télécharger la librairie Eigen
- Inclure le path de la librairie dans le répertoire Include (clic droit sur le projet -> propriétés -> Répertoires VC++ -> Répertoires Include)
